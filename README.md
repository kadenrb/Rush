Project: Rush Puzzle Game (Name pending change)

This is a grid based puzzle game built in Unity focusing on custom physics constraints and modular level design.
Some Key Technical Features

    Custom Movement Controller: Implemented a SweepTest based movement system to handle frame-perfect collisions in tight grid spaces.

    Modular Level Loading: Designed a GameController that handles level transitions via Prefab instantiation, allowing for rapid level iteration.

    Adaptive Snapping: Developed a snapping algorithm that handles different ship orientations (Horizontal/Vertical) and sizes (1x1 vs 1x3) to ensure a consistent player experience.
